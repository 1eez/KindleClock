# KindleClock

> 你的 Kindle 还好么？是否还藏在角落吃灰呢？赶紧拿出来擦一擦，与其闲置着不如让它成为一台墨水屏时钟，既实用又有格调。

🚀 **在线预览**: [https://clock.izhizhe.cn/](https://clock.izhizhe.cn/)

## 简介

KindleClock 是一个专为 E-ink 墨水屏设备（特别是旧款 Kindle）打造的极简时钟。
它采用了“复古”的技术栈（原生 ES5 + Table 布局），确保在 10 年前的浏览器内核上也能完美运行，流畅且省电。

### 核心特性
- **极致兼容**：零依赖，纯原生 JS，支持古董级 WebKit 内核。
- **墨水屏优化**：纯黑白高对比度设计，整点自动反色刷新防残影。
- **天气支持**：集成实时天气显示（自动定位），支持白天/夜间图标切换。
- **灵活设置**：
  - **双模式**：支持数字时钟 / 指针时钟。
  - **任意旋转**：0° / 90° / 180° / 270° 旋转，横竖屏通吃。
  - **强制刷新**：内置清除缓存刷新按钮，应对 Kindle 强势缓存。

## 运行机制 (Refresh Philosophy)

KindleClock 采用最省电的“**局部更新**”策略：
1.  **时钟刷新**：每 1 秒 (`1s`) 更新一次 DOM。页面**永不**整体重载。
2.  **天气刷新**：每 20 分钟 (`20min`) 自动后台请求一次数据。
3.  **防残影**：每小时整点（分秒均为 0）自动反色闪烁一次。

## 如何使用 😛

1.  打开 Kindle 的体验版浏览器。
2.  输入网址：`https://clock.izhizhe.cn/`。
3.  点击屏幕右上角（隐形按钮），打开设置面板调整方向和时间。

## 一点技巧 🥳

为了获得最佳体验，请配合以下 Kindle 秘籍使用：

### 1. 保持屏幕常亮 (`~ds`)
为了不让 Kindle 自动锁屏，请在 Kindle 主页（不是浏览器）的搜索栏输入以下命令并回车：

```text
~ds
```
输入后并不会有明显反应（也不会搜索出结果），但此时你尝试按电源键，会发现已经无法锁屏，即生效成功。
*想要恢复正常锁屏，长按电源键重启 Kindle 即可。*

### 2. 解决布局偏差
如果在浏览器中发现时钟没有居中，或者出现了滚动条，请尝试：
*   **双指捏合页面**（缩小）。
*   或者 **双击页面**。
这通常能重置浏览器的缩放状态，让绝对居中布局生效。

### 3. 如何部署
本项目无需构建。
直接将 `kindle-clock-vue` 目录下的 `index.html`, `src/` 文件夹上传至您的静态网站服务器根目录即可。
