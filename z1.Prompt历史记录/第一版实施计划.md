# KindleClock 实施计划

本项目旨在为 Kindle Paperwhite 3 (7代) 打造一个简洁、美观、功能可扩展的时钟系统。基于 Vue 3 和 Vite，遵循极小包体积和高性能的原则。

## 用户审核项
> [!IMPORTANT]
> - **隐藏设置入口**：点击屏幕 **右上角区域** 将触发设置界面的显示。
> - **全屏刷新**：为了防止墨水屏残留（Ghosting），时钟每小时会自动进行一次全屏黑白闪烁。
> - **技术方案**：采用 Vue 3 + Vite，打包构建目标设为 `es2018` 以保证在 Kindle 5.16.x 固件上的兼容性。

## 验证计划

### 自动化测试
- 运行 `npm run build` 验证构建后的包体积。要求：JS (gzip) ≤ 80KB，CSS ≤ 10KB。

### 手动验证
- **浏览器预览**：在 Chrome 的 Kindle PW 仿真分辨率（1072x1448）下检查排版美观度。
- **功能切换**：验证点击右上角是否能成功唤出设置菜单。
- **模式切换**：验证数字/指针时钟的切换是否顺畅。
- **部署验证**：由于暂时无法在实体 Kindle 上调试，我将提供一个本地测试流程，建议用户在 Kindle 浏览器中打开本地服务器地址查看效果。
